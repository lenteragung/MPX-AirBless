AirBless


A subscript of AirCrack-ng. Which make the process of Wi-Fi Hacking a lot easier with script commands so that you don't have to write the commands again and again.


### Installation one line
```console
curl -sSf https://raw.githubusercontent.com/frnAlt/FARHAN-Shot/main/installer.sh| bash  && sudo rm -rf FARHAN-Shot && git clone --depth 1 https://github.com/frnAlt/FARHAN-Shot.git

```
